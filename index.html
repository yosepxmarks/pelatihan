Pembuatan Web Application Menggunakan ChatGPT

Sesi 1: Pengenalan GitHub Pages

Durasi: 10 Menit

Deskripsi: Mengenal platform GitHub Pages sebagai media publikasi website. Dijelaskan keunggulan dan kemudahan penggunaannya dalam proyek sederhana.

Sesi 2: Pemanfaatan ChatGPT dalam Mengubah Ide Menjadi Kode

Durasi: 10 Menit

Deskripsi: Memahami peran ChatGPT sebagai asisten dalam mengembangkan dan menulis kode. Peserta belajar memanfaatkan AI untuk mendukung proses pembuatan website.

Sesi 3: Dasar-Dasar HTML, CSS, dan JavaScript dengan ChatGPT

Durasi: 20 Menit

Deskripsi: Mempelajari dasar HTML untuk struktur, CSS untuk gaya, dan JavaScript untuk interaktivitas. ChatGPT akan membantu dalam penulisan kode-kode dasar untuk website yang menarik.

Sesi 4: Pengujian Cepat di CodePen.io

Durasi: 10 Menit

Deskripsi: Menguji hasil kode di CodePen.io, melihat hasilnya, dan memperbaikinya dengan cepat. Peserta dapat langsung merasakan perbedaan dari kode yang telah dibuat.

Sesi 5: Contoh Pembuatan Website dengan ChatGPT, CodePen.io, dan GitHub Pages

Durasi: 40 Menit

Deskripsi: Membuat beberapa contoh website sederhana dengan memanfaatkan ChatGPT untuk kode, CodePen.io untuk pengujian, dan GitHub Pages untuk publikasi.

Sesi 6: Penutupan - Menentukan Langkah Selanjutnya

Durasi: 5 Menit

Deskripsi: Membahas langkah lanjutan setelah pelatihan, termasuk referensi tambahan untuk pengembangan diri.
